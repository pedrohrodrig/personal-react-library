# Alert

Alert component with customizable props for different severities and styles.

#### Example

<Canvas of={Alert.Component} />

## Usage

```tsx
import { Alert } from 'sld-ui';

const Example = () => {
  return (
    <Alert severity="success" variant="filled">
      This is a success alert!
    </Alert>
  );
};

export default Example;
```

#### Arguments

- **children** _`ReactNode`_ - The content of the alert.
- **severity** _`'success' | 'warning' | 'info' | 'danger'`_ - The severity
  level of the alert.
- **variant** _`'filled' | 'outlined'`_ - The style variant of the alert.
- **fixed** _`boolean`_ - Whether the alert is fixed (non-dismissible).
- **icon** _`boolean`_ - Adds or remove the icon.
