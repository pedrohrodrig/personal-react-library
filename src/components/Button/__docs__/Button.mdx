# Button

Button component with different props.

#### Example

<Canvas of={Button.Component} />

## Usage

```ts
import { Button } from "sld-ui";

const Example = () => {
  return (
    <Button
      size="small"
      onClick={() => console.log("Clicked")}
      variant="primary"
      fullWidth={false}
    >
      Button
    </Button>
  );
};

export default Example;
```

#### Arguments

- **children** _`React.ReactNode`_ - The content of the button.
- **variant** _`'primary' | 'secondary' | 'outline' | 'ghost'`_ - The variant of
  the button.
- **disabled** _`boolean`_ - A boolean indicating whether the button should be
  disabled or not.
- **size** _`'small' | 'medium' | 'large'`_ - The size of the button.
- **fullWidth** _`boolean`_ - A boolean indicating whether the button should
  take the full width of its container.
- **onClick** _`() => void`_ - A function that is called when the button is
  clicked.
